# 西方占星深度解读写作指南

本指南不是占星符号知识罗列，而是写作规范。目标是让 Kimi 输出类似专业占星师水准的深度本命星盘分析报告。

## 快速检查清单（解读前必读）

- [ ] 报告共 8 章，不可跳过任何章节
- [ ] 太阳、月亮、上升各展开 6 个维度
- [ ] 内行星按组合分析，不孤立罗列
- [ ] 三王星重点解读宫位而非星座
- [ ] 相位按能量互动解读，不做吉凶标签
- [ ] 关键相位逐一分析（每对不少于 2 句话）
- [ ] 推运盘 Transit 分析当前年份及未来 1-2 年
- [ ] 无"必然""注定""一定"等断言式词汇
- [ ] 结尾强调仅供文化娱乐参考

## 核心原则

### 1. 组合分析优先于单星分析
不要孤立地罗列"太阳在XX座、月亮在XX座"，而要分析**组合效应**：
- "太阳摩羯（目标导向）+ 月亮巨蟹（情感需求）" → 外在事业心强，内在渴望家庭温暖，需平衡事业与情感
- "水星天蝎（深度思维）+ 金星天秤（和谐审美）" → 思考深刻但表达追求平衡，适合谈判、调解、深度咨询
- "火星白羊（行动力）+ 土星摩羯（自律）" → 执行力强且有持久力，但可能过于压抑冲动
- "天王星合相太阳（变革力）+ 海王星合相月亮（梦幻感）" → 理想主义与现实变革的交织，人生充满戏剧性转折

### 2. 核心三元（太阳、月亮、上升）必须展开到6个维度
太阳、月亮、上升是本命盘的三根支柱，不能只说星座名称，必须深入分析每个维度的具体表现。

### 3. 相位是动力，不是标签
相位（合、冲、刑、拱、六分）描述的是行星之间的能量互动，不是吉凶标签：
- **合相（0°）**：能量融合，特质强化。可能过度也可能升华。
- **冲相（180°）**：内在张力，需要整合对立。张力越大，驱动力越强。
- **刑相（90°）**：内在冲突，需要调整。冲突点也是成长点。
- **拱相（120°）**：天赋顺畅，能量自然流动。但可能过于舒适而缺乏动力。
- **六分相（60°）**：机会与潜力，需主动激活。

### 4. 三王星（天海冥）是世代星，宫位比星座更重要
- 天王星、海王星、冥王星在同一星座停留多年，影响一整代人。
- 三王星对本命盘的影响主要体现在**它们落入的宫位**（人生领域），而非星座（性格特质）。
- 只有当三王星与个人行星（日月水金火）形成紧密相位时，才直接影响个人性格。

### 5. 每个论点必须配"具体表现"
不要说"你太阳摩羯所以事业心强"，而要说：
- "太阳摩羯座落在第十宫（事业宫），你对事业成就有强烈追求，渴望获得社会认可。但第十宫的土星特质也让你对'成功'的定义较为传统，可能需要拓宽对成就的认知。"

### 6. 趋势性描述，避免断言
- ❌ "你必然会成功" → ✅ "太阳第十宫+木星拱相，事业上有扩张机遇，但需配合努力与时机"
- ❌ "你注定孤独" → ✅ "月亮第十二宫可能让你倾向于隐藏情感需求，建议通过创作或心理咨询疏解"

## 报告结构（必须严格遵循）

### 一、基本信息

必须包含：
- 公历出生日期、时刻、地点、性别
- 上升星座、中天星座（MC）
- 太阳、月亮、水星、金星、火星、木星、土星、天王星、海王星、冥王星的落座与落宫
- 主要相位列表（合、冲、刑、拱、六分，容许度<5°）

### 二、核心三元解读（灵魂章节，必须展开到6个维度）

太阳、月亮、上升各6个维度，每个维度1-2句具体描述。

#### 2.1 太阳星座（人格核心、生命目标）

- **基本气质**：太阳星座的核心特质（如"摩羯座：务实、目标导向、追求成就"）
- **外在表现**：别人眼中的你（如"给人可靠、有责任感、专业严谨的印象"）
- **内在驱动力**：你最核心的生命目标（如"渴望通过事业成就获得社会认可与尊重"）
- **优势天赋**：太阳带来的先天优势（如"规划能力强，执行力高，擅长长期目标管理"）
- **潜在挑战**：太阳的阴暗面（如"可能过于功利，忽视情感需求，对自己要求过高"）
- **成长方向**：如何发挥太阳能量（如"学会在追求目标的过程中享受过程，而非只看结果"）

#### 2.2 月亮星座（内在情感需求、安全感来源）

- **基本气质**：月亮星座的核心情感模式（如"巨蟹座：情感细腻、渴望归属、保护欲强"）
- **情感需求**：你内心真正需要的是什么（如"需要家庭温暖、情感安全感、被照顾与被理解"）
- **安全感来源**：什么让你感到安心（如"熟悉的环境、亲密的陪伴、家庭氛围"）
- **情绪模式**：面对压力时的情绪反应（如"压力下倾向于退缩、情绪化、寻求情感庇护"）
- **潜在挑战**：月亮的阴暗面（如"过度依赖情感认同，情绪波动大，容易怀旧"）
- **成长方向**：如何滋养月亮（如"学会独立满足情感需求，建立内在安全感，不过度依赖他人"）

#### 2.3 上升星座（人格面具、第一印象、应对世界的方式）

- **外在形象**：你给人的第一印象（如"上升天蝎座：给人神秘、深沉、有穿透力的印象"）
- **应对模式**：面对新环境和新人的本能反应（如"先观察、试探，再决定是否深入交往"）
- **人生路径**：上升星座代表的人生剧本（如"上升天蝎：人生经历较多转化与重生，通过危机获得成长"）
- **优势**：上升带来的社交优势（如"有洞察力，善于挖掘真相，让人信任"）
- **挑战**：上升的面具可能带来的局限（如"过于防备，让人觉得难以接近，错失机会"）
- **整合**：如何让上升与太阳/月亮和谐（如"用天蝎的洞察力服务摩羯的事业目标，同时照顾巨蟹的情感需求"）

### 三、行星分布（必须组合分析，不能孤立）

#### 3.1 内行星（水、金、火）——思维模式、爱与价值、行动风格

- **水星**：思维与沟通方式
  - 不要只说"水星天蝎思考深刻"
  - 要分析：思维模式是什么？如何影响决策？如何影响沟通？如何影响学习？
  - 如"水星天蝎座：思维具有穿透力，善于挖掘本质，对表象不感兴趣。适合研究、侦探、心理学。但沟通中可能过于直接，让人感觉被冒犯。建议学会用委婉方式表达深刻见解。"

- **金星**：爱情观、价值观、审美
  - 不要只说"金星天秤爱美"
  - 要分析：你容易被什么样的人吸引？你的价值观是什么？你的审美倾向是什么？
  - 如"金星天秤座：对爱情追求和谐与平衡，容易被优雅、有品味的人吸引。价值观重视公平与美感，在关系中追求平等。审美偏向经典、优雅、对称。但可能过于追求和谐而回避冲突。"

- **火星**：行动力、欲望、冲突模式
  - 不要只说"火星白羊行动力强"
  - 要分析：你的行动力风格是什么？如何表达愤怒？如何追求目标？
  - 如"火星白羊座：行动力直接、果断，想到就做，不拖泥带水。愤怒来得快去得也快，不记仇。追求目标时勇往直前，但可能缺乏耐心，易半途而废。建议培养持久力，避免冲动决策。"

#### 3.2 外行星（木、土）——社会性与世代影响

- **木星**：扩张、机遇、信仰
  - 木星落座：你的信仰方向是什么？
  - 木星落宫：你在哪个领域容易获得机遇？
  - 如"木星射手座（入庙）：天生乐观，对哲学、宗教、旅行有兴趣。木星落在第九宫：高等教育、远行、跨文化领域是你的机遇所在。"

- **土星**：限制、责任、成长课题
  - 土星落座：你的恐惧是什么？
  - 土星落宫：你在哪个领域需要承担责任？
  - 如"土星摩羯座（入庙）：对责任与成就有深刻恐惧，害怕失败。土星落在第十宫：事业领域是你的成长课题，需要学会在压力下坚持，把恐惧转化为动力。"

#### 3.3 三王星（天海冥）——世代影响与个人化

- 三王星首先看**宫位**（人生领域），而非星座。
- 只有当三王星与个人行星（日月水金火）形成紧密相位（<3°）时，才详细分析对个人性格的影响。

**示例**：
> "天王星落在第五宫（创造力、恋爱、子女），你可能在恋爱和创作中追求独特性，不喜欢传统模式。天王星与太阳形成紧密合相（<2°），这意味着你天生有变革精神，不甘于现状，人生可能经历多次重大转折。"

### 四、宫位领域重点解读（必须结合主宰行星）

对每个重点宫位（至少解读5-7个宫位），分析：
- 宫头星座（此宫位的人生风格）
- 宫内行星（此领域的具体影响）
- 主宰行星（此宫头星座的守护星）落在哪里？代表什么？

**重点宫位**：
- **第一宫（命宫）**：自我形象、外在气质（结合上升星座）
- **第二宫（财富）**：财富观、物质安全感、赚钱方式
- **第四宫（家庭）**：家庭根源、内心安全感、父母影响
- **第五宫（恋爱/创造力）**：恋爱模式、创造力、子女
- **第七宫（婚姻/合作）**：婚姻模式、伴侣特质、合作关系
- **第八宫（深层资源/转化）**：他人资源、转化、心理深度
- **第十宫（事业）**：事业成就、公众形象、社会地位
- **第十二宫（潜意识/业力）**：潜意识、隐秘之事、灵性

**解读模板**：
> "第十宫（事业）宫头摩羯座，主宰行星土星落在第六宫。这意味着你的事业追求务实、有野心，愿意通过日常努力工作积累成就。但土星在第六宫也提示：事业成功需要建立在扎实的日常基础上，不要好高骛远。"

### 五、相位格局（必须识别关键相位组）

#### 5.1 关键相位分析

对主要相位（合、冲、刑、拱）逐一分析，每个相位说明：
- 涉及哪两颗行星？
- 相位的性质（合/冲/刑/拱）？
- 对命主的影响是什么？（天赋？张力？挑战？）
- 如何运用或转化？

**示例**：
> "太阳（摩羯座）与月亮（巨蟹座）形成冲相（180°）：这是你的核心张力。外在追求事业成就（太阳摩羯），内在渴望情感归属（月亮巨蟹）。你可能在事业与家庭之间反复拉扯。转化建议：不要让两者对立，而是让事业成就为家庭提供安全感，让家庭温暖为事业提供情感支撑。"

#### 5.2 相位格局识别（若构成）

- **大三角**（三颗行星互成120°）：天赋顺畅，能量自然流动。但可能过于舒适而缺乏挑战动力。
- **T三角**（两颗冲+一颗刑）：内在张力，驱动力强。刑相的顶点是突破方向。
- **大十字**（四颗行星两两冲刑）：多重压力，也蕴含巨大潜能。需要在压力中整合。
- **风筝**（大三角+一个对冲点）：既有天赋又有挑战。对冲点提供成长动力。
- **Yod**（两个行星六合一顶点，顶点与两者均成150°）：命运转折点。顶点是需要做出重大选择的领域。

> 每个格局必须结合命主实际情况给出具体影响，不能只列格局名称。

### 六、推运盘（Transit）简要分析（当前年份及未来1-2年）

本命盘是"地图"，推运盘（Transit）是"当前天气"。通过分析行星当前位置与本命盘的互动，可以了解当前及未来1-2年的运势走向。

#### 6.1 主要推运行星影响

**外行星推运（影响周期长，1-3年）**：
- **土星推运**：土星大约每2.5年经过一个星座。当土星经过本命盘某个宫位或与本命行星形成相位时，该领域会面临考验与成长。
  - 如"土星当前经过第十宫：事业上压力增大，但也正是奠定根基的关键期"
  - 如"土星与本命太阳形成刑相：自我认同面临挑战，需重新定义人生目标"

- **天王星推运**：天王星大约每7年经过一个星座。当天王星经过本命盘某个宫位或与本命行星形成相位时，该领域会发生突发变化或觉醒。
  - 如"天王星当前经过第七宫：感情/合作关系可能面临突变，需保持开放心态"
  - 如"天王星与本命金星形成合相：爱情观可能发生剧变，旧有关系模式被打破"

- **海王星推运**：海王星大约每14年经过一个星座。带来迷茫、灵性觉醒或创造力爆发。
  - 如"海王星当前经过第十二宫：潜意识活跃，适合灵性探索，但需防逃避现实"

- **冥王星推运**：冥王星运行极慢，影响一代人。只有当冥王星与本命个人行星（日月水金火）形成紧密相位时，才对个人产生深刻影响。
  - 如"冥王星与本命水星形成刑相：思维模式面临深刻转化，旧有认知被颠覆"

**内行星推运（影响周期短，几天到几周）**：
- 内行星（日月水金火）的推运影响较短，可作为日常参考，但本命盘分析中不必详细展开。

#### 6.2 太阳回归（Solar Return）简要说明

每年生日前后，太阳回到出生时的位置，形成太阳回归盘。太阳回归盘可以看作该年度的"年度星盘"，反映这一年的主题。

- 太阳回归盘的上升星座：该年度的整体基调
- 太阳回归盘中太阳所在的宫位：该年度的重点领域
- 如"太阳回归盘上升摩羯座：此年度基调务实、目标导向，适合规划长期目标"
- 如"太阳回归盘太阳在第十宫：此年度事业是重点，可能有升职或事业转折"

> 推运盘分析是简要参考，本命盘分析才是核心。推运盘提示当前机遇与挑战，但本命盘的天赋与课题才是根本。

### 七、综合人生策略（升华部分）

#### 7.1 人生全局气象
用1-2段话概括命主的整体人生基调：
- 如"你的星盘呈现出'太阳第十宫+月亮第四宫'的轴线，人生核心课题是在事业成就与家庭归属之间找到平衡。"
- 指出人生主旋律："先稳后变，中年发力"或"早年探索，中年扎根，晚年自由"等

#### 7.2 最突出的天赋（3-5项，每项配星盘依据）
- 天赋是什么？
- 星盘依据（哪个行星/宫位/相位）？
- 如何在现实中发挥？

#### 7.3 最需警醒的课题（2-3项，每项配转化建议）
- 是什么课题？
- 星盘依据（哪个刑相/落陷/挑战相位）？
- 如何转化/避免？（必须给出具体可操作的建议）

### 八、三维度策略建议（必须具体可操作）

#### 8.1 事业维度
- 适合行业（结合第十宫、第六宫、水星、土星）
- 关键年龄段（结合土星回归、天王星对冲等周期）
- 决策陷阱（如"避免木星过度扩张时的盲目乐观"）

#### 8.2 情感维度
- 感情模式（结合月亮、金星、第七宫、第五宫）
- 相处原则（具体行为建议）
- 关键时间节点（结合金星回归、土星回归）

#### 8.3 成长维度
- 性格修炼（结合月亮挑战相位、土星课题）
- 成长路径（如"发挥太阳领导力+培养月亮共情能力"）
- 精神追求（结合第九宫、第十二宫、木星、海王星）

## 写作质量检查清单

输出前检查是否满足以下要求：

- [ ] 太阳、月亮、上升各展开6个维度（基本气质、外在表现/情感需求/应对模式、内在驱动力/安全感来源/人生路径、优势天赋、潜在挑战、成长方向）
- [ ] 水星、金星、火星有组合分析，不是孤立罗列（结合落宫+相位）
- [ ] 木星、土星有宫位分析（机遇领域/成长课题）
- [ ] 三王星有宫位分析（人生领域），与个人行星有相位时才详细分析性格
- [ ] 重点宫位有宫头星座+宫内行星+主宰行星分析
- [ ] 关键相位有具体解读（不是只列角度）
- [ ] 相位格局有具体展开（不是只列名称）
- [ ] 综合总结有天赋+课题+建议
- [ ] 三维度策略建议具体可操作
- [ ] 全文没有"必然""注定""一定"等断言式词汇
- [ ] 全文保持积极正向，没有恐吓性预言
- [ ] 强调自由意志（星盘是地图，不是宿命）
