# 七政四余深度解读写作指南

本指南不是星历知识罗列，而是写作规范。七政四余极度依赖精密星历，本指南假设为示意级分析，但必须明确声明精度限制。

## 快速检查清单（解读前必读）

- [ ] 报告共 8 章，不可跳过任何章节
- [ ] 开头必须放精度声明（示意级，建议专业软件复核）
- [ ] 命度主分析含躔宿、状态、落宫、具体影响
- [ ] 身度主分析含内在需求、后天方向、与命度主关系
- [ ] 日月与五星分析含躔宿、状态、是否受克
- [ ] 四余星分析含对性情/业力的深层影响
- [ ] 当前大限及未来 2 个大限段落级解读
- [ ] 无"必然""注定""一定"等断言式词汇
- [ ] 结尾强调仅供文化娱乐参考

## 核心原则

### 1. 精度声明必须放在最前面
> **郑重声明**：本排盘为示意级简化版本。七政（日月金木水火土）使用 `ephem` 天文库计算，四余星（罗睺、计都、月孛、紫炁）使用 `pyswisseph` 精密星历计算；二十八宿采用传统宿度并岁差修正，十二宫按上升点黄道十二等分。七政四余极度依赖精密星历，不同流派算法差异大，星曜躔宿、庙旺喜乐、受克情况可能存在偏差。强烈建议使用专业七政四余排盘软件获取精确命盘后复核。以下内容仅供文化娱乐参考。

### 2. 组合分析优先于单星分析
不要孤立地罗列"太阳在XX宫、月亮在XX宫"，而要分析**组合效应**：
- "太阳躔XX宿（旺）+ 月亮躔XX宿（庙）" → 日月并明，内外光明，父母缘佳
- "木星躔XX宿（喜乐）+ 金星躔XX宿（庙）" → 木金相生，才华与魅力兼具，人缘好
- "火星躔XX宿（受克）+ 土星同宫" → 火土交战，行动力受阻，易有意外伤害或炎症
- "罗睺（计都）躔命宫" → 四余星对性情、业力的深层影响，可能带来前世业力或特殊缘分

### 3. 庙旺喜乐与受克必须影响解读强度
- **庙（入庙）**：星曜力量最强，正面特质发挥充分。解读时肯定语气。
- **旺（旺相）**：力量次强，正面为主。解读时以正面为主。
- **喜乐**：星曜喜欢此位置，表现自然。解读时客观正面。
- **落陷/受克**：星曜力量弱或受其他星克制，负面特质凸显。解读时指出挑战，给出转化方向。

### 4. 二十八宿度不可忽略
七政四余的精细度在于宿度，分析时必须指出星曜所躔的宿度，以及宿度对该星曜的影响：
- 如"太阳躔角宿（木宿）→ 太阳得木气之生，光明更盛，主贵"
- 如"火星躔参宿（水宿）→ 火入水宿受克，行动力易有挫折"

### 5. 每个论点必须配"具体表现"
不要说"你太阳好"，而要说：
- "太阳躔XX宿入庙，主贵气，外在形象光明磊落，有领导风范，易得男性长辈或上司提携"
- "月亮躔XX宿落陷，主内心敏感，情绪易波动，对安全感需求较高，需注意母系或女性关系"

### 6. 趋势性描述，避免断言
- ❌ "你必然会当官" → ✅ "太阳庙旺躔官禄宫，有领导潜质，但需配合大运与后天努力"
- ❌ "你注定短命" → ✅ "火星受克躔疾厄宫，需注意炎症与意外伤害，日常宜多运动疏解"

## 报告结构（必须严格遵循）

### 一、基本信息与精度声明

必须包含：
- 公历出生日期、时刻、地点、性别
- 真太阳时换算说明（若适用）
- **七政四余精度声明**：如上所述，必须放在报告开头
- 命度主、身度主（命主星、身主星）

### 二、命宫与命主（外在气质与人生趋向）

#### 2.1 命度主分析（核心）

- **命度主是什么星**：七政（日月金木水火土）中哪颗星是命度主？
- **命度主躔宿**：躔哪一宿？宿的五行属性（角亢氐房心尾箕东方青龙木、斗牛女虚危室壁北方玄武水等）
- **命度主状态**：庙旺喜乐？受克受制？
- **命度主落宫**：落在哪一宫？宫位对命主的影响

**解读模板**：
> "命度主为XX（星名），躔XX宿（XX行），状态为XX（庙旺/落陷/受克）。此星为命主带来了XX的先天特质：外在气质上XX（具体表现），人生趋向上XX（适合什么方向），但也需注意XX（潜在挑战）。"

**示例**：
> "命度主为太阳，躔角宿（木宿），入庙旺相。太阳得木气之生，光明更盛。此命主外在气质光明磊落，有领导风范，易得男性长辈或上司提携。人生趋向宜走公众事业、管理、政治路线。但太阳过旺也需注意不要过于自我，学会倾听他人意见。"

#### 2.2 命宫其他星曜

命宫中的其他星曜对命主的影响：
- 日月躔命宫：主贵，但需看日月是否反背
- 金星躔命宫：主貌美、人缘、艺术天赋
- 火星躔命宫：主刚毅、冲动，需注意情绪管理
- 木星躔命宫：主仁慈、博学、贵人缘
- 水星躔命宫：主聪明、善辩、学习能力强
- 土星躔命宫：主稳重、谨慎，但也可能保守、压抑

### 三、身宫与身主（内在需求与后天着力点）

#### 3.1 身度主分析

- **身度主是什么星**：与命度主可能不同
- **身度主躔宿**：躔哪一宿？状态如何？
- **身度主落宫**：落在哪一宫？代表后天着力方向

**解读模板**：
> "身度主为XX（星名），躔XX宿（XX行），状态为XX。身度主揭示了命主内在的核心需求：中年后运势重心转向XX（方向），后天着力宜在XX领域深耕。身度主与命度主的关系：XX（如一致则内外和谐，如矛盾则早年与中年追求不同）。"

**示例**：
> "身度主为月亮，躔心宿（火宿），状态平和。月亮入心宿，内在需求是情感安全与精神归属感。中年后运势重心转向家庭、情感、内心修养。与命度主太阳相比，早年追求外在成就，中年更注重内在平衡，人生会出现从'外求'到'内守'的转化。"

#### 3.2 身宫星曜组合

身宫中的星曜组合对中年后的影响：
- 如"身宫有木星+金星：中年后贵人缘佳，适合从事文化、艺术、教育领域"
- 如"身宫有火星+罗睺：中年后需注意情绪与冲动，业力课题可能浮现"

### 四、七政四余分布（重点分析星格组合）

#### 4.1 日月分析（核心）

- **太阳**：代表父亲、上司、外在形象、男性力量。躔宿、状态、落宫。
- **月亮**：代表母亲、女性、内在情绪、女性力量。躔宿、状态、落宫。
- **日月关系**：
  - 日月同宫或夹拱：日月并明，内外光明，父母缘佳
  - 日月反背：太阳落陷+月亮落陷，内心矛盾，外在表现与内在需求不一致
  - 日月相刑冲：父母关系可能不睦，或自身内外有冲突

#### 4.2 五星分析（金木水火土）

- **金星**：美貌、人缘、爱情、艺术。躔宿、状态、落宫。是否受克？
- **木星**：仁慈、学问、贵人、扩张。躔宿、状态、落宫。是否受克？
- **水星**：智慧、口才、交易、学习。躔宿、状态、落宫。是否受克？
- **火星**：行动、勇气、冲动、炎症。躔宿、状态、落宫。是否受克？
- **土星**：稳重、责任、压抑、拖延。躔宿、状态、落宫。是否受克？

#### 4.3 四余星分析（深层影响）

四余星（紫炁、月孛、罗睺、计都）对性情、业力的深层影响：
- **罗睺**：南交点，代表前世业力、执着、欲望。躔命宫：有特殊使命感或前世未了课题。
- **计都**：北交点，代表今生成长方向、业力转化。躔命宫：今生需突破的课题。
- **紫炁**：代表清高、精神追求、超脱。躔命宫：有出世倾向，对物质看淡。
- **月孛**：代表情欲、桃花、暗昧。躔命宫：感情丰富，但需注意桃花劫。

#### 4.4 星格组合（必须识别）

- **日月夹拱**：日月夹拱某星，该星力量大增，主该领域有贵人或天赋
- **五曜连珠**：五星聚集某宫，主该领域人生有重大机遇或挑战
- **星入本垣**：某星躔入其本宿（如太阳躔房日兔），力量最强
- **星入庙旺**：某星躔入庙旺宿，正面特质发挥
- **星受克**：某星被其他星克制或躔入克宿，需注意该星代表领域的挑战

### 五、洞微大限（当前大限及近期运势）

#### 5.1 当前大限分析

- **走到哪一宫**：大限宫位的主星是什么？
- **大限主星状态**：庙旺？落陷？受克？
- **大限四余影响**：罗睺、计都等是否在此大限产生作用？
- **近期运势主线**：这步大限的整体基调（起步/突破/考验/稳定）

**解读模板**：
> "当前处于XX大限（XX岁-XX岁），大限主星为XX（星名），躔XX宿，状态XX。此大限的整体基调为XX（起步/突破/考验/稳定）。近期运势主线：XX（具体方向）。机遇：XX。挑战：XX。建议：XX。"

### 六、领域解读（按宫位切入）

#### 6.1 官禄宫（事业）
- 官禄宫主星、躔宿、状态
- 官禄宫与命宫的联动（如"命宫太阳+官禄宫木星：事业光明且贵人多"）
- 事业方向与关键期建议

#### 6.2 财帛宫（财富）
- 财帛宫主星、躔宿、状态
- 求财方式与财运特征
- 理财建议

#### 6.3 夫妻宫（婚姻情感）
- 夫妻宫主星、躔宿、状态
- 配偶特质与感情模式
- 婚姻建议与关键时间节点

#### 6.4 迁移宫（外出发展）
- 迁移宫主星、躔宿、状态
- 外出发展利弊
- 异地机遇与人际关系

#### 6.5 其他宫位（简要）
- 父母宫、兄弟宫、子女宫、疾厄宫、田宅宫、奴仆宫
- 每个宫位一句话概括，涉及重点宫位详细展开

### 七、综合总结（升华部分）

#### 7.1 人生全局气象
用1-2段话概括命主的整体人生基调：
- 如"你的七政四余命盘呈现出'太阳入庙、木星得地'的格局，先天贵气与学识兼备，人生基调光明稳健。"
- 指出人生主旋律："早年得父辈提携，中年靠自己积累，晚年享清福"等

#### 7.2 最突出的天赋（3-5项，每项配命理依据）
- 天赋是什么？
- 命理依据（哪个星/宿/组合）？
- 如何在现实中发挥？

#### 7.3 最需警醒的课题（2-3项，每项配转化建议）
- 是什么课题？
- 命理依据（哪个星落陷/受克/四余影响）？
- 如何转化/避免？（必须给出具体可操作的建议）

### 八、三维度策略建议（必须具体可操作）

#### 8.1 事业维度
- 适合行业（结合命度主和官禄宫）
- 关键大限（结合洞微大限）
- 决策陷阱（如"避免火星受克时盲目扩张"）

#### 8.2 情感维度
- 感情模式（结合夫妻宫和月孛/金星）
- 相处原则（具体行为建议）
- 关键时间节点（结合大限）

#### 8.3 成长维度
- 性格修炼（结合四余星业力课题）
- 成长路径（如"发挥太阳贵气+培养月亮共情"）
- 精神追求（适合学习什么、修行什么）

## 写作质量检查清单

输出前检查是否满足以下要求：

- [ ] 报告开头有精度声明（示意级，建议专业软件复核）
- [ ] 命度主分析有躔宿、状态、落宫、具体影响（不少于4个维度）
- [ ] 身度主分析有内在需求、后天方向、与命度主的关系
- [ ] 日月分析有躔宿、状态、关系（同宫/夹拱/反背/刑冲）
- [ ] 五星分析每颗星都有躔宿、状态、落宫、是否受克
- [ ] 四余星分析有对性情、业力的深层影响
- [ ] 星格组合有具体展开（不是只列名称）
- [ ] 洞微大限有段落级详细解读（不少于当前大限+未来2个）
- [ ] 领域解读有官禄、财帛、夫妻、迁移（至少4个宫位）
- [ ] 综合总结有天赋+课题+建议
- [ ] 三维度策略建议具体可操作
- [ ] 全文没有"必然""注定""一定"等断言式词汇
- [ ] 全文保持积极正向，没有恐吓性预言
- [ ] 庙旺喜乐/落陷受克的状态影响了解读强度（不是一概而论）
