# 八字深度解读写作指南

本指南不是知识罗列，而是写作规范。目标是让 Kimi 输出类似人类命理师水准的深度分析报告，而非干瘪的排盘结果。

## 快速检查清单（解读前必读）

- [ ] 报告共 14 章，不可跳过任何章节
- [ ] 日主画像 6 个维度，每个维度至少 4 个具体表现点
- [ ] 十神组合按年柱、月柱、日柱、时柱逐一展开，每柱至少 3 个具体表现
- [ ] 每个论点配具体场景，避免空泛标签
- [ ] 关键大运段落级解读（每个不少于 150 字）
- [ ] 健康提醒按重点领域逐一展开，含症状、诱因、建议
- [ ] 三维度策略每个维度至少 6 条具体行为
- [ ] 无"必然""注定""一定"等断言式词汇
- [ ] 结尾强调仅供文化娱乐参考

## 核心原则

### 1. 组合分析优先于单星分析
不要孤立地罗列"年柱劫财、月柱食神"，而要分析**组合效应**：
- "偏印月令 + 食伤双透" → 深度思考 + 才华表达，但易想多做少
- "日支偏财 + 劫财透年干" → 有商业嗅觉但竞争压力大
- "土旺 + 金日主" → 才华被压抑，需木疏通

### 2. 每个论点必须配"具体表现"
不要说"你很有才华"，而要说：
- "你擅长把复杂想法转化为他人能理解的内容（偏印深度 + 食伤表达）"
- "你对质感、美感、细节有天然鉴赏力（辛金特质 + 偏印专注）"

### 3. 趋势性描述，避免断言
- ❌ "必然离婚" → ✅ "31-41岁大运感情面临考验，需用心经营"
- ❌ "注定贫穷" → ✅ "财星受土包围，求财过程易有波折，需主动疏通"
- ❌ "一生无贵人" → ✅ "贵人运需靠主动积累人脉，而非天然贵人扶持"

### 4. 每个维度必须给出"建设性建议"
分析完问题后，必须给出可操作的解决方案，不能停留在"你有这个问题"的层面。

## 报告结构（必须严格遵循）

### 一、基本信息

必须包含：
- 公历出生日期、出生时刻、出生地点、性别
- **真太阳时换算说明**：若经度与120°E差异>15分钟，必须注明换算过程（如"巫山109.88°E，时差约-40分钟，北京时间09:00对应真太阳时约08:20"）
- 农历日期
- 真太阳时与北京时间的时柱差异对比（如有）

### 二、八字四柱排盘

表格格式：

| 柱 | 天干 | 地支 | 地支藏干 | 藏干十神 | 天干十神 |
|----|------|------|----------|----------|----------|
| **年柱** | X | X | 藏干 | 十神 | 十神 |
| **月柱** | X | X | 藏干 | 十神 | 十神 |
| **日柱** | X | X | 藏干 | 十神 | 比肩（日主） |
| **时柱** | X | X | 藏干 | 十神 | 十神 |

> 若有北京时间/真太阳时时柱差异，需单独注明对比。

### 三、日主与格局分析（这是整个报告的灵魂）

#### 3.1 日主画像（必须展开6个维度，每个维度至少4个具体表现点）

不要只说"甲木为阳木"，要按以下维度展开，每个维度给出**4-6个具体场景化的表现**（不是泛泛而谈）：

**1. 先天禀赋与性格底色**
- 日主五行的核心特质（如辛金"珠玉之金，外柔内刚"）
- 月令对日主的影响（如"生于未月，土旺金埋，才华待疏"）
- 日柱干支关系对性格的塑造（如"辛卯日柱，卯木偏财坐日支，对美感有天然感知"）
- 具体表现（至少4点）：
  - 外柔内刚：表面温和，内心有清晰原则
  - 审美敏锐：对色彩、设计、音乐有天然感知力
  - 细节控：易察觉他人忽略之处
  - 思虑深远：三思而后行，但有时会错失良机
  - 喜清不喜浊：对环境和人际有较高洁净度要求

**2. 思维模式与信息处理**
- 月柱对思维的影响（如"月柱癸未，食神坐偏印，才华输出有阻滞"）
- 时柱对思维的影响（如"时柱壬辰，伤官坐正印，创新被收束"）
- 具体表现（至少4点）：
  - 直觉与逻辑并重：既能数据驱动分析，也能凭直觉判断
  - 创意发散但落地难：想法多但缺乏落地能量
  - 喜辩论但不喜争吵：享受逻辑博弈，厌恶情绪化冲突
  - 对抽象与哲学感兴趣：偏印格对形而上问题有好奇心

**3. 情感模式与亲密关系**
- 日支对感情的影响（如"日支卯木偏财，对感情有感官体验需求"）
- 十神对感情的影响（如"食神伤官透出，从朋友发展为恋人"）
- 具体表现（至少4点）：
  - 被智慧型、艺术型异性吸引
  - 容易从朋友发展为恋人
  - 感情中需要空间：独立需求高，需要定期独处
  - 对旧感情有执念倾向：分手后反复回溯

**4. 事业驱动力与职业倾向**
- 格局对事业的影响（如"偏印格用食伤，思想家+表达者双重潜质"）
- 喜用神对事业的指引（如"喜木火，适合创意策划、文化审美"）
- 具体表现（至少4点）：
  - 适合创意策划、文化审美、技术钻研类工作
  - 在团队中常是冷面智囊角色
  - 对不公正敏感：适合规则透明的团队或自主创业
  - 职业变动有周期性：财富机会呈阶段性爆发

**5. 社交模式与人际关系**
- 年柱对社交的影响（如"年柱庚金劫财，朋友圈质量参差不齐"）
- 十神对社交的影响（如"伤官食神透出，社交场合不怯场"）
- 具体表现（至少4点）：
  - 交游广泛但深交不多
  - 容易与朋友发生利益冲突
  - 贵人多为长辈或权威
  - 网络社交活跃

**6. 身心状态与能量周期**
- 最旺/最弱五行对身心的影响（如"土旺金埋，思虑多行动少"）
- 具体表现（至少4点）：
  - 思虑型失眠：夜深人静时头脑停不下来
  - 消化系统敏感：饮食不规律时易胃胀反酸
  - 肩颈僵硬：久坐后气血不畅
  - 春夏精神、秋冬低迷：能量有季节性波动

> **关键要求**：每个维度必须包含**具体的生活场景描述**，不能只说"你有这个特质"，而要说"你在购物时会反复对比参数"、"当伴侣向你抱怨时你的第一反应是分析原因"等。**每个维度至少4个具体表现点**。

#### 3.2 格局判断（必须分三层递进）

- **第一层：月令定格局**
  - 月令本气是什么？对日主是生/克/泄/耗？
  - 如"未月己土本气，对辛金为偏印，故为偏印格"

- **第二层：天干透出的组合效应**
  - 天干透出哪些十神？它们之间形成什么关系？
  - 如"年干庚金（劫财）、月干癸水（食神）、时干壬水（伤官）食伤双透，形成'食伤泄秀'之势"

- **第三层：整体格局的"生泄并见"特点**
  - 印星生身 + 食伤泄身 → 有才华需要表达，但印星也赋予深度思考倾向
  - 不要只列格局名称，要解释格局对命主人生的实际影响

### 四、五行旺衰与喜用神（必须有推导过程）

#### 4.1 五行力量统计表格

| 五行 | 力量 | 来源 |
|------|------|------|
| 金 | X | 天干X、藏干X |
| 木 | X | 藏干X |
| 水 | X | 天干X、藏干X |
| 火 | X | 藏干X |
| 土 | X | 藏干X、X |

#### 4.2 旺衰判断（必须解释为什么）

- **日主旺衰**：月令是否生扶？有无比劫帮身？是否有泄耗？
  - 如"辛金生于未月（土旺），地支两辰一未土重，土多金埋，幸有庚金劫财帮身，日主中和偏旺"

- **最旺/最弱五行**：指出哪个五行最旺、哪个最弱，并说明对命局的影响
  - 如"土最旺：土重金埋，易有才华被压抑、想法多但行动滞后的倾向"
  - 如"火最弱：仅未中藏一丁火，火力不足，需补火调候"

#### 4.3 调候用神与格局喜用神（必须区分）

八字中**用神**有两层含义，必须分别说明：

**一、调候用神（解决寒热燥湿）**
- 调候用神与日主旺衰无关，只看出生月份的气候。
- 夏生（巳午未月）多热燥，需水调候；冬生（亥子丑月）多寒湿，需火调候。
- 如"辛金生于未月（夏末），天气炎热，需水调候降燥，否则金气过燥难以成器"
- 如"甲木生于子月（冬月），天寒地冻，需火调候暖局，否则木气冻僵难以生发"

**二、格局喜用神（解决旺衰扶抑）**
- 根据日主旺衰和格局需求确定。
- 日主强：喜克泄耗（官杀、食伤、财星）
- 日主弱：喜生扶（印枭、比劫）
- 如"辛金日主中和偏旺，土旺埋金，喜木疏土、喜火炼金成器，忌土金"

**三、两者关系**
- 若调候用神与格局喜用神一致，则最佳（如夏生金日主，调候需水，格局也喜水泄秀）。
- 若两者冲突，需综合权衡：通常**调候优先于格局**，因为气候条件决定命局的根本环境。
- 如"冬生甲木，调候需火，但格局若火多木焚，则火不可过旺，需水木并济"

#### 4.4 纳音五行参考

年柱、日柱的纳音可作为辅助参考：
- 纳音五行与正五行（天干地支五行）不同，属于另一套五行体系。
- 年柱纳音代表先天根基与祖业背景，日柱纳音代表个人气质与婚姻特质。
- 如"庚辰年柱纳音白蜡金：先天根基为精致之金，主早年有家教，但需火炼方能成器"
- 如"辛卯日柱纳音松柏木：个人气质坚韧不屈，婚姻中需有水土滋养"

> 纳音五行作为辅助参考，不可喧宾夺主。主要分析仍以正五行（天干地支五行）为准。

#### 4.5 特殊格局：从格判断（简要参考）

若日主极旺或极弱，需考虑从格：
- **从旺格（从强格）**：日主极旺，无克制，满盘皆是生扶。宜顺势而为，喜用神为生扶之五行。
  - 判断：日主得月令+比劫+印星重重，无比肩劫财之外的克制。
- **从弱格（从儿/从财/从杀）**：日主极弱，无生扶，满盘皆是克泄耗。
  - 从儿格：食伤极旺，日主无根，喜用神为食伤所生之五行（财）。
  - 从财格：财星极旺，日主无根，喜用神为财星。
  - 从杀格：官杀极旺，日主无根，喜用神为官杀。
- **判断要点**：月令+地支是否有根？天干是否有比劫/印星帮身？若全无根气，则考虑从格。

> 从格较为罕见，大部分命局为正格（扶抑格）。若不确定，按正格分析即可，不必强行从格。

| 喜忌 | 五行 | 说明 |

| **喜神** | X | 如"调候暖局，炼金成器，生旺财星" |
| **可用** | X | 如"食伤泄秀，但水多土荡，需适度" |

| **忌神** | X | 如"比劫争财，竞争加剧" |

> **核心建议**：给出后天调节方向（颜色、方位、行业、习惯等），必须具体可操作。

### 五、十神组合与性格特质（必须分析组合，不能孤立）

#### 5.1 核心十神组合分析（按柱位逐一展开 + 总论）

不要只提取3-5个组合，而是**对年柱、月柱、日柱、时柱分别进行十神组合详解**，最后加一段**四柱十神组合总论**。

**年柱十神组合详解**：
- 年柱天干+地支的十神组合（如"年柱庚辰：劫财+正印"）
- 年柱代表的早年运势（1-15岁）
- 具体表现：
  - 早年家庭环境的影响（如"父母管教较严，有文化底蕴"）
  - 同辈竞争或兄弟姐妹的影响（如"劫财透干，早年感受到竞争压力"）
  - 祖辈背景对性格的塑造（如"辰为正印，家族有传统背景"）

**月柱十神组合详解**：
- 月柱天干+地支的十神组合（如"月柱癸未：食神+偏印"）
- 月柱为提纲，对命局影响最大
- 具体表现：
  - 才华与表达的模式（如"食神被偏印克制，想法多但表达不畅"）
  - 母亲或女性长辈的影响（如"母亲关爱方式带有控制感"）
  - 青少年时期的核心体验（如"青春期有叛逆与探索的交织"）

**日柱十神组合详解**：
- 日柱天干+地支的十神组合（如"日柱辛卯：比肩+偏财"）
- 日柱代表自己与配偶
- 具体表现：
  - 个人核心特质（如"对财富和感情有近距离追求"）
  - 配偶特质与婚姻模式（如"配偶有才华但可能不稳定"）
  - 财务管理倾向（如"有赚快钱机会但储蓄能力一般"）

**时柱十神组合详解**：
- 时柱天干+地支的十神组合（如"时柱壬辰：伤官+正印"）
- 时柱代表晚年、子女、事业最终形态
- 具体表现：
  - 晚年生活状态（如"伤官佩印，叛逆转化为有章法的创新"）
  - 子女特质与教育方式（如"子女有才华但可能难以管教"）
  - 事业最终形态（如"适合独立咨询或教学工作"）

**四柱十神组合总论**（必须有一段总结）：
- 四柱十神组合的整体效应（如"劫财+伤官+食神组合，表达欲+竞争欲+创新欲"）
- 能量波动特征（如"情绪与能量状态是波动性的，高能量期文思泉涌，低能量期沉默寡言"）
- 修炼重点（如"让表达更接地气，让思维更有温度"）

> **关键要求**：每个柱位的分析不少于3个具体表现点。总论段落不少于100字。

#### 5.2 性格画像（用一段概括+ bullet points）

**概括段**：用1段话总结核心性格基调（如"你是一个外柔内韧、才华横溢但偶有纠结的人"）。

**优势**：列出3-5个具体优势，每个配一句解释。
**挑战**：列出2-3个具体挑战，每个配一句解释和转化建议。

### 六、事业财运分析（必须具体到行业、理由、关键期）

#### 6.1 事业方向表格

| 适合领域 | 具体理由（必须引命理依据） |
|----------|---------------------------|
| 领域一 | 如"食伤泄秀，审美敏锐，适合设计、写作、影视、艺术" |
| 领域二 | 如"食伤主表达，偏印主学识，适合教学、培训、咨询" |
| 领域三 | 如"辛金主精工，偏印主专研，适合技术、工艺、研发" |
| 领域四 | 如"偏财坐日支，有商业嗅觉，适合市场、品牌、策划" |
| 领域五 | 如"伤官透时干，不喜约束，适合独立工作、自媒体、创业" |

**注意**：每个领域必须给出**具体的命理依据**（引哪个十神、哪个五行），不能只说"适合做这个"。

#### 6.2 财运特征

- 财星位置和状态（日支偏财？天干正财？）
- 财星是否受克？（如"卯木被辰未土包围，财星受克"）
- 比劫对财的影响（如"劫财透干，合伙需谨慎"）
- 求财建议（必须具体：如"依靠专业技能和个人品牌赚钱，避免高风险投机"）

#### 6.3 事业发展关键期（大运视角）

对**当前大运**和**未来2-3个关键大运**进行段落级解读：

- **年龄段**：XX岁-XX岁（XX运）
- **十神组合**：XX + XX，代表什么？
- **运势基调**：是起步期、突破期、还是稳定期？
- **具体建议**：这个阶段应该做什么、避免什么？

**示例**：
> **21-31岁（丙戌大运）**：正官+正印，事业起步期，适合进入正规平台或体制内发展，奠定职业基础。丙火正官为喜神，对你有利。但戌土为燥土，仍有"土旺埋金"之忧，需要主动寻找木火属性的机会来疏通。**建议**：趁此运奠定事业根基，不要急于求成，稳扎稳打为上。

### 七、感情婚姻分析（必须具体到模式和伴侣类型）

#### 7.1 感情模式

- **日支看配偶特质**：日支是什么？对命主是什么十神？
  - 如"日支卯木为偏财（男命妻星）：对伴侣有较高期待，容易被温柔、有艺术气质或经济头脑的异性吸引。"

#### 10.3 流年分析（当前年份及未来1-2年）

- **流年干支**：如"2024年甲辰流年"
