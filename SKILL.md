# 命理占星综合分析

## 前置声明

任何输出开头必须加入：

> **郑重声明：本文由AI基于命理知识库生成，仅供文化娱乐与自我探索，所有内容不可作为人生决策依据。**

## 信息收集

执行前必须确认用户提供以下信息。若缺失，礼貌询问：

1. **公历出生日期**（年月日，农历需特别注明）
2. **出生时刻**（精确到分钟，如 09:30）
3. **出生地点**（省/市/区县，用于确定真太阳时）
4. **性别**（男/女，用于排大运）

## 工作流程

1. **确认信息**：补齐缺失字段，询问用户希望分析的体系（八字/紫微/七政四余/西方占星/全部）。
2. **时间换算**：出生地点经度与北京时间（120°E）差异超过15分钟时，必须换算为真太阳时后再排盘。换算公式：`真太阳时 = 北京时间 + (经度 - 120) × 4分钟`。例如：巫山（109.88°E）与北京时差约 -40分钟，北京时间09:00 对应真太阳时约08:20。八字排盘以真太阳时为准，同时注明北京时间差异。
3. **排盘计算**（读取对应 JSON 输出文件，关键字段如下）：
   - **八字** `bazi_result.json`
     - 调用：`python scripts/bazi_pan.py <年> <月> <日> <时> <分> <性别> [城市]`
     - 关键字段：`pillars`（四柱）、`canggan`（藏干）、`shishen`（十神）、`wuxing_count`（五行统计）、`shensha`（神煞）、`dayun`（大运）、`advanced.pattern`（格局）、`advanced.tiaohou`（调候用神）、`advanced.xiyongshen`（喜用神）
   - **紫微斗数** `ziwei_result.json`
     - 调用：`python scripts/ziwei_pan.py <年> <月> <日> <时> <分> <性别> [城市]`
     - 关键字段：`ming_gong`（命宫）、`shen_gong`（身宫）、`wuxing_ju`（五行局）、`gong_layout`（十二宫地支）、`gong_stars`（各宫主星/辅星/四化/庙旺）、`daxian`（大限）
   - **七政四余** `qizheng_result.json`
     - 调用：`python scripts/qizheng_pan.py <年> <月> <日> <时> <分> <性别> [城市]`
     - 关键字段：`ming_shen`（命宫/身宫、命度主/身度主）、`qizheng`（七政躔宿）、`siyu`（四余星）、`daxian`（洞微大限）、`today_xiu`（当日值宿）
   - **西方占星** `western_result.json`
     - 调用：`python scripts/western_pan.py <年> <月> <日> <时> <分> <性别> <城市>`
     - 关键字段：`big_three`（太阳/月亮/上升）、`planets`（行星分组）、`houses`（宫位分组）、`aspects`（相位分级）、`elements`（元素统计）、`modalities`（模式统计）、`patterns`（格局识别）

   **异常处理**：
   - 脚本返回非零：检查 `<gender>` 是否为 `男`/`女`，城市是否支持；如仍失败，跳过该体系并在报告中说明。
   - `city` 无法 geocode：脚本会使用默认经纬度（北京 116.4°E, 39.9°N），你应在报告中注明「城市未识别，使用默认坐标」。
   - 依赖缺失（`cnlunar`/`ephem`/`pyswisseph` 等）：脚本会降级到简化计算并在 `stderr` 输出警告，你应在报告开头声明「某某体系因缺少依赖使用简化算法，结果仅供示意」。
4. **分体系解读**：按需加载 `references/` 中对应指南，进行深度解读。
5. **交叉综合**：加载 `references/cross-summary-guide.md`，整合多体系核心信息。
6. **输出报告**：以 Markdown 格式输出完整文档，分章节清晰呈现。

## 排盘工具

所有脚本基于 `cnlunar`（农历/八字）、`ephem`（天文计算）、`geopy`（地理位置）、`pyswisseph`（精密星历/真实 Placidus 分宫）实现，支持真太阳时自动换算。

| 工具 | 路径 | 用途 | 输出文件 | 精度 |
|------|------|------|----------|------|
| 基础转换 | `scripts/lunar_convert.py` | 公历转农历、节气、四柱干支、真太阳时、城市坐标、二十八宿、神煞、纳音、西方占星基础（行星/宫位/相位） | `lunar_convert_result.json` | 1901-2100年精确 |
| 八字排盘 | `scripts/bazi_pan.py` | 八字四柱、藏干、十神、大运、五行统计、神煞、**格局判断、调候用神、喜用神** | `bazi_result.json` | 1901-2100年精确 |
| 紫微斗数排盘 | `scripts/ziwei_pan.py` | 安命宫/身宫、五行局、十四主星、辅星、四化、庙旺利陷、**十二宫布局、大限** | `ziwei_result.json` | 基于 cnlunar 农历数据 |
| 西方占星排盘 | `scripts/western_pan.py` | 太阳/月亮/行星位置、**上升星座、十二宫位、宫位主星、相位** | `western_result.json` | 基于 ephem 地心黄道坐标 |
| 七政四余排盘 | `scripts/qizheng_pan.py` | 七政躔宿、四余星、**二十八宿、洞微大限** | `qizheng_result.json` | 七政用 `ephem`，四余星用 `pyswisseph` 精密星历；仍属示意级 |

**脚本调用示例**：
```bash
# 八字排盘（可选指定城市用于真太阳时）
python scripts/bazi_pan.py 1990 5 15 14 30 男 北京

# 紫微斗数排盘
python scripts/ziwei_pan.py 1990 5 15 14 30 男 北京

# 西方占星排盘
python scripts/western_pan.py 1990 5 15 14 30 男 北京

# 七政四余排盘
python scripts/qizheng_pan.py 1990 5 15 14 30 男 北京

# 指定输出文件路径
set FORTUNE_OUTPUT_FILE=D:\result.json && python scripts/bazi_pan.py 1990 5 15 14 30 男 北京
```

**依赖安装**：
```bash
pip install cnlunar ephem geopy pyswisseph
```

## 重要注意事项

- **外部库依赖**：脚本依赖 `cnlunar`（农历/八字/神煞）、`ephem`（天文计算/行星位置）、`geopy`（城市坐标查询）、`pyswisseph`（精密星历/真实 Placidus 分宫）。首次使用前需运行 `pip install cnlunar ephem geopy pyswisseph`。若库未安装，脚本会降级到简化计算并输出警告。**Windows 环境提示**：如果 `python` 或 `pip` 不在 PATH 中，可使用 `where python` / `where pip` 查找完整路径，然后用完整路径执行安装（如 `C:\Python314\python.exe -m pip install cnlunar ephem geopy pyswisseph`）。
- **七政四余**：`qizheng_pan.py` 使用 `ephem` 计算七政位置（日月金木水火土），`pyswisseph` 计算四余星（罗睺、计都、月孛、紫炁）；二十八宿采用传统宿度并岁差修正，十二宫按上升点黄道十二等分。仍属示意级，建议用户用专业排盘软件复核。
- **紫微斗数**：`ziwei_pan.py` 基于 cnlunar 农历数据实现安星算法，覆盖命宫、主星、四化、大限等核心要素。安星步骤繁复，建议用户使用专业排盘软件复核。
- **西方占星**：`western_pan.py` 使用 ephem 计算地心黄道坐标，支持行星位置、上升星座、宫位、相位。默认使用 Placidus 分宫；安装 `pyswisseph` 时调用真实 Placidus 算法，未安装时回退到近似插值。
- **真太阳时**：所有脚本支持通过 `city` 参数自动查询城市经纬度并换算真太阳时。出生地点经度与北京时间（120°E）差异超过15分钟时，会自动换算。换算公式：`真太阳时 = 北京时间 + (经度 - 120) × 4分钟`。例如：巫山（109.88°E）与北京时差约 -40分钟，北京时间09:00 对应真太阳时约08:20。八字排盘以真太阳时为准，同时注明北京时间差异。
- **立春与节气**：年柱以立春为界，月柱以节气（节）为界，严格遵循。
- **23时规则**：23:00-23:59出生，日柱与时柱均按次日计算。
- **脚本输出编码**：排盘脚本结果写入 UTF-8 编码文件，请勿依赖 stdout 中文输出（可能存在终端编码问题）。通过读取输出文件获取完整中文结果。
- **占位符替换**：`references/` 指南中的 `XX`（如「太阳躔XX宿」「命度主为XX」）是模板占位符，解读时必须用命盘真实数据替换，不得原样输出。
- **娱乐性质**：反复强调本内容仅供文化娱乐，禁止作为医疗、投资、婚姻等重大决策依据。
